# Savings Tracker

A simple and intuitive web application to track your savings goals and monthly deposits.

## Features

- **Goal Tracking**: Create and manage multiple savings goals.
- **Monthly Insights**: Visualize your monthly deposits with a dynamic chart.
- **Import/Export**: Easily backup or restore your data using JSON files.
- **Responsive Design**: Built with Bootstrap for a seamless experience across devices.

## Project Structure

- `assets/`: Contains static resources (CSS, Images).
- `js/`: Contains the JavaScript modules and application logic.
- `index.html`: The main entry point of the application.

## Getting Started

Simply open `index.html` in your browser to start tracking your savings!

## Technologies Used

- HTML5 & CSS3
- JavaScript (ES6 Modules)
- [Bootstrap 5](https://getbootstrap.com/)
